#### 53 求某个区间的最大和

求取区间和用前缀差

分别用了三种方法：

* O(n*n) 暴力遍历所有可能解

* 每个i求最大值：当前sum值-前面最小前缀和，和最大值 比较，并且维护一个前面的最小前缀值

* 动态规划

  动态规划获得 以i为结尾的最大数组和

  那么d[0] d[1] ....d[i]中最大值就是所求值
